# 🏨 Great God Guest House Hotel Management System (HMS)

A modern web-based Hotel Management System designed to simplify hotel operations including room management, reservations, guest records, bookings, payments, staff administration, and reporting.

---

## 📌 Project Overview

This application provides hotel staff with a centralized dashboard to manage day-to-day hotel activities.

The system is designed to support:

- Room Inventory Management
- Guest Registration
- Reservations
- Bookings & Check-ins
- Check-outs
- Payment Processing
- Revenue Tracking
- Staff Management
- Reports & Analytics
- Hotel Settings

The frontend is built using HTML, CSS, and JavaScript while the backend will be integrated separately.
